# 第二节：时间膨胀（Time Dilation）

## 2.1 光钟实验

### 装置设计
- **光钟**：两面镜子相距 D，光在两面镜子间弹跳
- 静止时一个周期的时间间隔：Δt = 2D/c
- 这是一个**极简装置**，便于分析运动时的行为

### 四步推理法（贯穿全书的推理框架）

1. **构建装置**于静止参考系中
2. **加载装置**到运动参考系（Alice 的车中）
3. **用光速不变**计算 Bob 看到的 ΔtB
4. **推广结论**：根据 (R)，如果光钟的时间膨胀，那么所有钟——手机、心率——都应有相同的时间膨胀

## 2.2 时间膨胀的推导

### 垂直方向长度不变性
- **思想实验**：火车与铁轨
  - 火车静止时，车轮间距 = 铁轨宽度
  - 火车运动时，车轮间距不会变大或变小（否则会导致不同类型的事故，违反 (E)）
- **结论**：垂直于运动方向的长度在所有参考系中相同（D⊥ 不变）

### 推导 ΔtB

- Alice 看到：Δ tA = 2D/c
- Bob 看到的光路径（直角三角形）：斜边 = c(ΔtB/2), 底边 = v(ΔtB/2), 高 = D
- 勾股定理：D² + (v ΔtB/2)² = (c ΔtB/2)²
- 解得：**ΔtB = γ ΔtA**，其中 **γ ≡ 1/√(1 - v²/c²)**

### γ 因子的性质
- v = 0 → γ = 1（无时间膨胀）
- v → c → γ → ∞（时间趋近停止）
- v = 0.995c → γ ≈ 10

## 2.3 双生子佯谬（Twin Paradox）

### 问题
- Bob（静止）认为 Alice（运动）更年轻
- 但运动是相对的 (R)，Alice 是否也应认为 Bob 更年轻？

### 解答（两种情况）

**情况一：双方都是惯性系**
- 两人以相对速度永远匀速分离
- 只能相遇**一次**（出发时）
- "Alice 认为 Bob 更年轻"和"Bob 认为 Alice 更年轻"都正确——**不矛盾**
- 因为没有第二次相遇来比较年龄

**情况二：Alice 返回（加速）**
- Alice 要返回必须经历**加速度**（非惯性）
- 在整个旅程中，Alice 的世界线不是直线（加速度打破对称性）
- Bob 始终是惯性系，Alice 不是——**对称性被打破**
- **结论**：返回的 Alice 确实比 Bob 年轻

## 2.4 时间旅行

- **去未来的时间旅行**：乘坐近光速飞船往返，返回时地球已过去多年
- **回到过去的时间旅行**：在狭义相对论框架内存在严重困难
  - 因果悖论（杀死祖父、遇见另一个自己等）
  - 很可能不可能，但目前没有决定性结论

## 2.5 物理直觉的培养

### 三种解题方式
1. **模式匹配**：归结为已知问题，套用公式
2. **反向搜索**：从问题倒推所需公式（考试常用）
3. **物理图像引导**："在脑中放电影"——想象手榴弹飞行的慢动作，建立起 γ 因子的直觉

### 建立物理直觉的方法
- 在脑中对问题"放电影"，包含尽量多的物理细节
- 对不直观的问题反复思考
- 关注佯谬及其解决方式
- 从不同角度思考同一问题
- 与日常经验类比，找出相似和关键差异
- 简化和模块化问题

## 关键公式

| 公式 | 含义 |
|------|------|
| γ = 1/√(1 - v²/c²) | Lorentz 因子 |
| ΔtB = γ ΔtA | 时间膨胀：运动时钟变慢 |

## 要点总结

- 四步推理法是狭义相对论的核心方法论
- 垂直于运动方向的长度在所有参考系中一致（由 (E) 保证）
- 运动时钟确实变慢——这不是表象，而是真实的物理效应
- 双生子佯谬中，加速/返回的一方（世界线更弯曲）才是更年轻的那一方
