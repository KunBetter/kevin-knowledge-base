# 第三节：长度收缩与速度叠加（Length Contraction & Velocity Addition）

## 3.1 光尺（Light Ruler）实验

### 装置设计
- 一维"光尺"：发射器 → 镜子（距离 D）→ 检测器
- 静止时：Δt = D/c + D/c = 2D/c
- 光在单向传播中花一半时间，往返总时间为 2D/c

### 四步推理

1. **静止光尺**：DA = (c/2) ΔtA
2. **加载到 Alice 车上**：沿运动方向放置光尺
3. **Bob 的观测**：
   - 光去程：c ΔtB1 = DB + v ΔtB1 → ΔtB1 = DB/(c-v)
   - 光回程：c ΔtB2 = DB - v ΔtB2 → ΔtB2 = DB/(c+v)
   - 总时间：ΔtB = ΔtB1 + ΔtB2 = (2DB/c) / (1 - v²/c²) = (2/c) γ² DB
4. **推广**：对任何尺子都成立

### 推导长度收缩

- ΔtB = γ ΔtA（时间膨胀，已知）
- (2/c) γ² DB = γ (2/c) DA
- **DB = DA / γ**

**结论：运动尺子沿运动方向收缩，收缩因子为 1/γ。**

## 3.2 关于运动参考系中长度的说明

- DB 不是 Bob 自己的尺子长度
- 而是 **Alice 的尺子**（随 Alice 运动）在 Bob 参考系中的长度
- 对于 Bob（静止观察者），Alice 的运动尺子**收缩**了
- 历史名称：**洛伦兹收缩**（FitzGerald 1889, Lorentz 1892 提出，早于狭义相对论）

## 3.3 速度叠加公式

### 光尺变体测速仪

修改光尺：将发射到镜子的光替换为速度 u 的物体（从检测光改为接收 u 物体的反射信号）

### 推导

- 静止时（Alice 的参考系中）：Δ tA = D/uA + D/c
- Bob 参考系中（利用已知结论）：
  - ΔtB = γ ΔtA
  - DB = DA / γ
  - 物体去程：Δ tB1 = DB/(uB - v)
  - 光回程：ΔtB2 = DB/(c + v)
  - ΔtB = DB/(uB - v) + DB/(c + v)

### 结果

除以静止方程并化简，得到：

**uB = (uA + v) / (1 + uA v/c²)**

- 当 uA = c 时，uB = (c+v)/(1+v/c) = c ✓（与 (C) 一致）
- 当 v ≪ c 且 uA ≪ c 时，uB ≈ uA + v（退化为伽利略速度叠加）

### 迅速度（Rapidity）

定义 **φ(v) ≡ arctanh(v/c)**，则：

**φ(uB) = φ(uA) + φ(v)**

迅速度才是真正可叠加的变量！为什么出现双曲函数？预示了时空的双曲旋转结构（第 7 节答案）。

## 3.4 横向上也有速度变换

对任意方向的速度，利用洛伦兹变换可得：

| 方向 | 变换公式 |
|------|---------|
| x 方向 | vBx = (vAx + v) / (1 + vAx v/c²) |
| y 方向 | vBy = vAy / [γ(1 + vAx v/c²)] |
| z 方向 | vBz = vAz / [γ(1 + vAx v/c²)] |

**注意**：即使横向速度，Alice 和 Bob 测的值也不同——因为**时间间隔不同**，而长度间隔相同。

### 例子：垂直方向的速度叠加
当 vA = (0, u, 0)，即垂直于相对运动方向时：
vB = (v, u/γ, 0)

## 关键公式

| 公式 | 含义 |
|------|------|
| DB = DA / γ | 长度收缩 |
| uB = (uA + v)/(1 + uA v/c²) | 相对论速度叠加 |
| φ(uB) = φ(uA) + φ(v) | 迅速度叠加（可加性） |

## 要点总结

- 运动尺子沿运动方向收缩，收缩到 1/γ
- 垂直方向长度不变（由 (E) 保证，见第 2 节火车思想实验）
- 相对论速度叠加公式确保光速在任何参考系中保持 c
- 迅速度 φ = arctanh(v/c) 才是真正可加的"速度"
- 时间膨胀、长度收缩、速度叠加——三者相互关联，可递推
