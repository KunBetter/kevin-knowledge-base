# 第六节：洛伦兹变换（The Lorentz Transformation）

## 6.1 问题的提出

- Alice 和 Bob 有相对运动（v 沿 x 方向）
- 给定一个事件在 Bob 坐标系中的 (tB, xB)，如何得到 Alice 坐标系中的 (tA, xA)？
- 这相当于**坐标变换**——从 Bob 的"网格"到 Alice 的"网格"

## 6.2 旋转类比（时空-空间变换的类比）

### 纯空间旋转
- Alice 和 Bob 静止在同一个点，但面向不同方向（夹角 θ）
- 坐标变换：
  ```
  xA =  xB cosθ + yB sinθ
  yA = -xB sinθ + yB cosθ
  ```
- 保持不变的是：xA² + yA² = xB² + yB²（欧几里得不变量）

### 问题提示
- 洛伦兹变换将是类似的"旋转"形式——但在**时空**中而非空间中
- 不变的不是 x² + y²，而是 -c²t² + x²

## 6.3 洛伦兹变换的推导

### 已知条件
1. 长度收缩：DB = DA/γ → x 方向需包含 γ 因子
2. 垂直方向长度不变 → y 和 z 不变
3. 时间膨胀：Δ tB = γ ΔtA
4. 相对运动的对称性

### 推导过程

**空间部分**（从长度收缩出发）：
- xA = γ (xB - v tB)  —— 在 Alice 看来，Bob 的 xB 被压缩了
- xB = γ (xA + v tA)  —— 由 (R) 对称性得到

**时间部分**（从空间变换反推）：
- 将 xA = γ(xB - v tB) 代入 xB = γ(xA + v tA) 消去 xA
- 得到：tA = γ (tB - v xB / c²)
- 对称地：tB = γ (tA + v xA / c²)

## 6.4 完整的洛伦兹变换

```
ctA = γ (ctB - β xB)        ctB = γ (ctA + β xA)
xA  = γ (xB  - β ctB)       xB  = γ (xA  + β ctA)
yA  = yB                    yB  = yA
zA  = zB                    zB  = zA
```

其中 **β ≡ v/c**，**γ = 1/√(1 - β²)**

### 时间变换中的"空间偏移"
- tA = γ (tB - v xB / c²) 中的 **-v xB / c²** 项：
  - 即使 Bob 的时间 tB 相同，不同位置 xB 的事件的 Alice 时间也不同
  - **这就是同时性相对性的数学表述！**

## 6.5 洛伦兹对称性

> 洛伦兹变换表达了 Alice 和 Bob 之间的**对称性**：
> 尽管他们用不同的坐标描述事件，但他们观察到**相同的自然定律**

- "对称性"现在被认为是基础物理学的**第一原理**
- 洛伦兹变换形成了一个数学**群**——洛伦兹群（Lorentz Group）

## 6.6 一般方向的速度变换

对任意速度方向 vA = (vAx, vAy, vAz)，使用洛伦兹变换得到 Bob 看到的速度：

| 分量 | 变换公式 |
|------|---------|
| vBx | (vAx + v) / (1 + vAx v / c²) |
| vBy | vAy / [γ (1 + vAx v / c²)] |
| vBz | vAz / [γ (1 + vAx v / c²)] |

## 关键公式

| 公式 | 含义 |
|------|------|
| xA = γ (xB - v tB) | 空间坐标变换 |
| tA = γ (tB - v xB / c²) | 时间坐标变换 |
| β = v/c | 无量纲速度 |
| γ = 1/√(1-β²) | Lorentz 因子 |

## 要点总结

- 洛伦兹变换是连接不同惯性系的**完整坐标变换**
- 空间变换包含长度收缩，时间变换包含时间膨胀和同时性偏移
- 变换展示了时空的统一性——时间和空间坐标混合
- 不变量是 -c²t² + x² + y² + z²（而非欧几里得的 x²+y²+z²）
