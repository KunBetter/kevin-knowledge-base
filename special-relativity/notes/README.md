## 讲义特色

1. **思想实验驱动**：以"王二"的近光速飞船故事开篇，用故事化叙述展开物理概念
2. **四步推理法**：构建装置 → 加载到运动参考系 → 用光速不变计算 → 推广到所有装置
3. **物理直觉培养**：强调"在脑中放电影"，不只依赖公式推导
4. **历史脉络**：从伽利略、麦克斯韦、迈克尔逊-莫雷到爱因斯坦的完整科学史

## 核心知识点

1. **爱因斯坦两大假设**：相对性原理 (R) + 光速不变 (C) + 事件独立于观察者 (E)
2. **时间膨胀**：运动时钟变慢，ΔtB = γ ΔtA
3. **长度收缩**：运动尺子沿运动方向收缩，DB = DA / γ
4. **同时性的相对性**：不同观察者对"同时"的定义不同
5. **洛伦兹变换**：时空坐标的转换规则
6. **闵可夫斯基时空**：非欧几里得的时空几何，ds² = -c²dt² + dx²
7. **相对论动量与能量**：p = γmv, E = γmc², E² = m²c⁴ + p²c²
8. **质能等价**：E = mc²（静止能量）
