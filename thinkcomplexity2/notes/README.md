## 贯穿全书的主题

1. **涌现（Emergence）**：简单规则 → 复杂行为
2. **小世界现象**：高聚类 + 短路径
3. **重尾分布**：复杂系统的普遍特征
4. **解释的本质**：抽象模型 v.s. 详细模型 v.s. 数学证明
5. **计算等价性**：Class 4 系统与通用计算
6. **哲学反思**：实在论、工具主义、可证伪性
